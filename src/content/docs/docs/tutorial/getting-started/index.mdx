## Log in to ThingsBoard

Open the ThingsBoard web interface.
- If you are using a **local deployment**, sign in with the default credentials:
  - **System Administrator:** `sysadmin@thingsboard.org` / `sysadmin`
  - **Tenant Administrator:** `tenant@thingsboard.org` / `tenant`  
  You can change the password for each account on the profile page after login.
- If you are using **ThingsBoard Cloud**, sign in using Google, GitHub, Apple, or your email address.

After successful authentication, you will be redirected to the ThingsBoard interface.  
All further steps in this guide are performed at the **Tenant** level.
