title: User management
description: How to manage users in the ThingsBoard License Server portal — add, invite, and delete users.
---

import ImageGallery from '~/components/ImageGallery.astro';

A single account on the License Server may create unlimited users. The Users page allows you to invite other users to your account. Each user within a single account has access to both create new and edit existing subscriptions, as well as manage billing info.

To create a new user, press **Add User**, fill in the required fields, and send the **Activation link** to the user.

<ImageGallery images={[{ src: '/src/assets/images/license-server/users.png', alt: 'User management in License portal' }]} />

Within the User management page you may also log in as a particular user or delete the user.
