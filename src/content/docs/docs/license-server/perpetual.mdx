---
title: Perpetual fallback license
description: Definition of perpetual fallback license and how to purchase it via ThingsBoard License Server.
---

A perpetual fallback license is a license that allows you to use a specific version of software without an active subscription for it.
When purchasing a perpetual fallback license, you get one year of software updates included for eternal usage of ThingsBoard PE. After the first year, you can continue using the platform and purchase software updates for subsequent years at a discounted fee.

A single perpetual fallback license covers a single ThingsBoard PE instance (server process).
So, if you want to run ThingsBoard PE in an HA mode, you will need at least two licenses.

In order to purchase a Perpetual license, register on the License portal and then follow simple steps:

- Go to **Perpetual** → **Choose a product** → Input **Billing info** → **Create**

Or choose an exact product from [pricing](/pricing/). Once you press the **Get your license** button, you will be routed to the License portal authentication/registration, or directly to the **Billing info** step if you are already logged in on the License portal.
