This guide walks you through installing ThingsBoard Professional Edition from Azure Marketplace using the <a href="https://docs.microsoft.com/en-us/azure/marketplace/marketplace-faq-publisher-guide#pricing-and-payment" target="_blank">BYOL (Bring Your Own License)</a> model. You get the license directly from ThingsBoard, Inc, and purchase the server instances and infrastructure from Azure.

## Prerequisites

- Active <a href="https://azure.microsoft.com/en-us" target="_blank">Microsoft Azure</a> account

## Step 1. Subscribe to ThingsBoard PE BYOL

Open the <a href="https://azuremarketplace.microsoft.com/en-us/marketplace/apps/things-board.tb-pe-byol" target="_blank">ThingsBoard Professional Edition BYOL</a> product page on Azure Marketplace.

## Step 2. Configure virtual machine

You will be redirected to the "Create a virtual machine" dialog.

### 2.1 Basics

- Create a new **Resource group** (e.g. `thingsboard`)
- Set a recognizable **Virtual machine name** (e.g. `ThingsBoardPE-PROD`)
- Choose a **Region** from the available list
- Set **Administrator Account** to **SSH public key**. Username **must** be `ubuntu`. See <a href="https://docs.microsoft.com/en-us/azure/virtual-machines/linux/ssh-from-windows" target="_blank">Azure SSH docs</a> for key generation

## Step 4. Obtain your license secret

Register on the <a href="https://license.thingsboard.io/signup" target="_blank">ThingsBoard License Portal</a> to get your license. See <a href="https://www.youtube.com/watch?v=dK-QDFGxWek" target="_blank">How to get a pay-as-you-go subscription</a> or <a href="https://www.youtube.com/watch?v=GPe0lHolWek" target="_blank">How to get a perpetual license</a> for details.

Save your license secret — you will need it in the next step.

## Step 5. Configure your license secret

Connect to your instance over SSH using the <a href="https://docs.microsoft.com/en-us/azure/virtual-machines/linux/ssh-from-windows" target="_blank">official Azure guide</a>. Use the instance public IP from Step 3 and the SSH key from Step 2.1.

Open the configuration file:

```bash
sudo nano /etc/thingsboard/conf/thingsboard.conf
```

Locate the license configuration block and uncomment the export statement. Replace `YOUR_LICENSE_SECRET_HERE` with your actual license secret:

```bash
# License secret obtained from ThingsBoard License Portal (https://license.thingsboard.io)
# UNCOMMENT NEXT LINE AND PUT YOUR LICENSE SECRET:
export TB_LICENSE_SECRET=YOUR_LICENSE_SECRET_HERE
```

## Step 6. Launch ThingsBoard service

```bash
sudo service thingsboard start
```

<Aside type="note">
Allow up to 120 seconds for the Web UI to start on slower machines (1–2 CPUs or 1–2 GB RAM).
</Aside>

## Step 7. Connect to ThingsBoard UI

Open the ThingsBoard web interface in your browser using your instance's external IP address.

Default credentials:
- **Login**: sysadmin@thingsboard.org
- **Password**: sysadmin

## Post-installation steps

### Configure HTTPS

<Steps>
1. Assign a valid DNS record to your instance's external IP address.
2. Connect to your instance via SSH.
3. Run the following commands, replacing the domain and email with your own:

   ```bash
   sudo certbot-certonly --domain smth.yourcompany.com --email support@yourcompany.com
   sudo haproxy-refresh
   ```
</Steps>
