ThingsBoard source code is available under the <a href="https://github.com/thingsboard/thingsboard" target="_blank">Apache 2.0 license</a>. You can build the Community Edition from source and use it as a base for evaluation or development.

<Aside type="note">
ThingsBoard Professional Edition is distributed as pre-built packages and Docker images. Building PE from source is not supported.
</Aside>

See the <DocLink product={Products.CE} path="installation/building-from-source">CE Building from Sources</DocLink> guide for full instructions.
