This guide walks you through installing ThingsBoard Professional Edition from GCP Marketplace using the BYOL (Bring Your Own License) model. You get the license directly from ThingsBoard, Inc, and purchase the server instances and infrastructure from GCP.

## Prerequisites

- Active <a href="https://cloud.google.com/" target="_blank">GCP</a> account

## Step 1. Launch ThingsBoard PE BYOL

Open the <a href="https://console.cloud.google.com/marketplace/details/thingsboard-public/thingsboard-pe" target="_blank">ThingsBoard Professional Edition BYOL</a> product page on GCP Marketplace.

<ImageGallery images={[
  { src: '/src/assets/images/docs/installation/gcp-marketplace-pe/launch.png', alt: 'GCP Marketplace launch page', caption: 'Click "Launch on Compute Engine".' },
]} />

Click **Launch on Compute Engine**. You can use the default settings or customize them. When ready, click **Deploy**.

<ImageGallery images={[
  { src: '/src/assets/images/docs/installation/gcp-marketplace-pe/deploy.png', alt: 'GCP deployment configuration', caption: 'Configure settings and click "Deploy".' },
]} />

Once the deployment completes:

<ImageGallery images={[
  { src: '/src/assets/images/docs/installation/gcp-marketplace-pe/ssh.png', alt: 'GCP deployment complete', caption: 'Click "SSH" to open a terminal session.' },
]} />

Click **SSH** to open a browser-based SSH session. Keep this window open — you will use it in Step 3.

## Step 2. Obtain your license secret

Register on the <a href="https://license.thingsboard.io/signup" target="_blank">ThingsBoard License Portal</a> to get your license. See <a href="https://www.youtube.com/watch?v=dK-QDFGxWek" target="_blank">How to get a pay-as-you-go subscription</a> or <a href="https://www.youtube.com/watch?v=GPe0lHolWek" target="_blank">How to get a perpetual license</a> for details.

Save your license secret — you will need it in the next step.

## Step 3. Configure your license secret

Open the configuration file:

```bash
sudo nano /etc/thingsboard/conf/thingsboard.conf
```

Locate the license configuration block and uncomment the export statement. Replace `YOUR_LICENSE_SECRET_HERE` with your actual license secret:

```bash
# License secret obtained from ThingsBoard License Portal (https://license.thingsboard.io)
# UNCOMMENT NEXT LINE AND PUT YOUR LICENSE SECRET:
export TB_LICENSE_SECRET=YOUR_LICENSE_SECRET_HERE
```

## Step 4. Launch ThingsBoard service

```bash
sudo service thingsboard start
```

<Aside type="note">
Allow up to 120 seconds for the Web UI to start on slower machines (1–2 CPUs or 1–2 GB RAM).
</Aside>

## Step 5. Connect to ThingsBoard UI

Open the ThingsBoard web interface in your browser using your instance's external IP address.

<ImageGallery images={[
  { src: '/src/assets/images/docs/installation/gcp-marketplace-pe/admin-panel.png', alt: 'ThingsBoard admin panel', caption: 'ThingsBoard login page.' },
]} />

Default credentials:
- **Login**: sysadmin@thingsboard.org
- **Password**: sysadmin

## Post-installation steps

### Configure HTTPS

<Steps>
1. Assign a valid DNS record to your instance's external IP address.
2. Connect to your instance via SSH.
3. Run the following commands, replacing the domain and email with your own:

   ```bash
   sudo certbot-certonly --domain smth.yourcompany.com --email support@yourcompany.com
   sudo haproxy-refresh
   ```
</Steps>
