# Cloud (PaaS)

Welcome to the Cloud documentation. This is the fully managed cloud version of the platform.

## Key Features

- Fully managed infrastructure
- Automatic scaling
- Built-in monitoring
- No maintenance required
- Pay-as-you-go pricing

## Quick Start

Getting started with the Cloud version:

1. Create an account at the cloud portal
2. Set up your organization
3. Add devices and configure dashboards
4. Start collecting data
