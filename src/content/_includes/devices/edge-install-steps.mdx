## Starting Edge on the {props.deviceName}

<Steps>

1. Log in to the **ThingsBoard Professional Edition instance** and go to the **Edge Management > Instances** section. Click the **"+"** icon in the top right corner and select the **"Add new edge"** option.
2. Enter a name for your Edge in the **"Name"** field and click the **"Add"** button to confirm the addition of your new Edge.
3. Your new edge should now appear at the top of the list, as entries are sorted by creation time by default.

</Steps>

To initiate an **SSH (Secure Shell)** connection to the **{props.deviceName}**, open the Terminal and run the following command:

```bash
ssh USER@ip_address  # Enter the actual IP address
```

Replace `USER` with `{props.userName}` (the default user on this device) and `ip_address` with the actual IP address of the **{props.deviceName}**. The default password is `{props.password}`.

Once connected, follow the installation instructions below. Start by creating a new directory:

```bash
mkdir tb_edge
```

Open this directory:

```bash
cd ~/tb_edge
```

Create a docker compose file for the **ThingsBoard Edge** service within this directory:

```bash
nano docker-compose.yml
```

Then, you need to fill this file with the configuration lines. Once the **Edge** instance has been created, the preset installation instructions will be available for use. They contain important credentials, such as **Edge Secret**, **Edge Key**, etc. To access these configurations:

<Steps>

1. Go to the **Edge management > Instances** section of your **ThingsBoard Professional Edition** instance, and click on the **Instance**. Then, click the **"Install & Connect Instructions"** button.
2. On the **"Install & Connect Instructions"** pop-up window, select the **"Docker"** tab and **copy the configuration lines**.

</Steps>

Paste the copied lines into the **docker-compose.yml** file and press **CTRL+S** to save it. To close the file press **CTRL+X**. Then execute the following commands:

```bash
docker compose up -d
docker compose logs -f mytbedge
```

<Aside type="note">
The command must be executed in the **same directory** in which the `docker-compose.yml` file was saved.
</Aside>

To set up local port forwarding via SSH, open **another terminal tab** and run the following command:

```bash
ssh -N -L 8080:127.0.0.1:8080 USER@ip_address  # Enter the actual IP address
```

Any connection to **localhost:8080** on your local machine will be forwarded to **127.0.0.1:8080** on the **{props.deviceName}**.

The **ThingsBoard Edge** instance is available at [http://127.0.0.1:8080](http://127.0.0.1:8080). Use your credentials to log in.

Once Edge is running, continue with the rest of this guide and use `localhost` as your hostname when you reach the **Connect device** step.
