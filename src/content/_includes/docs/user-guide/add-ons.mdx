ThingsBoard offers add-on products that extend the core platform with specialized capabilities. Each add-on integrates tightly with the main ThingsBoard instance and can be deployed alongside it.

| Add-on | Description | Editions |
|--------|-------------|----------|
| **<DocLink product={props.product} path="user-guide/edge-computing">Edge Computing</DocLink>** | Run ThingsBoard at the edge — on gateways or local servers — for local data processing, offline operation, and seamless cloud synchronization | CE, PE, Cloud |
| **<DocLink product={props.product} path="user-guide/trendz-analytics">Trendz Analytics</DocLink>** | Advanced analytics engine for building prediction models and anomaly detection on top of ThingsBoard data | PE, Cloud |
