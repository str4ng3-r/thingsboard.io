In this step, you will create an Edge instance record in your ThingsBoard account, install the instance, and access the Edge UI.

---

## Step 1.1 Provision Edge

Before installing Edge software, create an Edge instance record in your ThingsBoard account. This generates the credentials Edge uses to authenticate.

<Steps>
1. Log in to your ThingsBoard account as a Tenant Administrator. Use your own credentials or the defaults (`tenant@thingsboard.org` / `tenant`).

2. Go to **Edge Management → Instances** and click **Add edge** (+).

3. Enter a name for the Edge instance and click **Add**.
</Steps>

## Step 1.2 Install ThingsBoard Edge

After you create the Edge instance, the installation instructions appear in a dialog. Each Edge has preset configurations that contain important credentials, such as **Edge Secret**, **Edge Key**, etc.

<Steps>
1. In the **Edge created** dialog, select your installation method — **Docker**, **Ubuntu**, or **CentOS** — and follow the displayed commands to install and start ThingsBoard Edge.

2. Once Edge starts (this may take a few seconds), open <a href="http://localhost:8080" target="_blank" rel="noopener noreferrer">http://localhost:8080</a> in a browser and log in with your Tenant Administrator credentials.
</Steps>
