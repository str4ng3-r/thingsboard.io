The **Entities Table** widget displays the latest values for a list of entities matching the selected <DocLink product={props.product} path="user-guide/aliases">alias</DocLink> and filter, with full-text search and pagination. You can customize its appearance through widget styles, data source keys, and <DocLink product={props.product} path="user-guide/actions">widget actions</DocLink>.

## Add an Entities Table widget

<Steps>
1. Enter dashboard edit mode by clicking the **pencil** icon in the lower right corner.
2. Click the **+** sign to add a new widget.
3. In the widget bundle selection dialog, choose **Cards**.
4. Select the **Entities table** widget.
5. Click **Add** to attach a datasource. A datasource consists of an entity alias and the telemetry keys to display.
6. Select an existing alias from the dropdown or click **Create a new one**. See <DocLink product={props.product} path="user-guide/aliases" bold={false}>Aliases</DocLink> for details on alias types.
7. Add telemetry keys from the dropdown.
8. Click **Add** to confirm.
9. Drag the edges of the widget to adjust its size. Click the **checkmark** to save.
</Steps>

### Row style function

Customize row styles based on entity values using a JavaScript function. The function receives the `entity` object and should return a CSS style object.

```js
result = { background: "white" };
if (entity.active == "false") {
   result.background = '#FF0000';
} else {
   result.background = '#00FF00';
}
return result;
```
