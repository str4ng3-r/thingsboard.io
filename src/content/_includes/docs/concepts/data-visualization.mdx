The Visualization Layer transforms raw telemetry into interactive control panels. Dashboards aggregate multiple data sources into a single pane of glass for real-time monitoring and remote hardware control.

- <DocLink product={props.product} path='user-guide/dashboards'>Dashboard:</DocLink> Centralized UI container for visualizing data and managing entities. Support multiple entities, states, and layouts.
- <DocLink product={props.product} path='user-guide/dashboards'>Dashboard State:</DocLink> Logical "page" or view within a dashboard. Used to build drill-down hierarchies, like **Building** asset → **Room** sensor.
- <DocLink product={props.product} path='user-guide/widgets'>Widget:</DocLink> Modular UI element used to display telemetry or execute commands: **Charts**, **Gauges**, **Maps**, **Tables**, etc.
- <DocLink product={props.product} path='reference/widgets/widget-library'>Widget bundle:</DocLink> Curated collection of widgets grouped by function: **Digital Gauges**, **Charts**, **Control Widgets**, etc.
