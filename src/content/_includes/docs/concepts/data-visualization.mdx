The Visualization Layer transforms raw telemetry into interactive control panels. Dashboards aggregate multiple data sources into a single pane of glass for real-time monitoring and remote hardware control.

- <DocLink product={props.product} path='user-guide/TBD/'>Dashboard:</DocLink> Centralized UI container for visualizing data and managing entities. Supports multi-page layouts and role-based access control.
- <DocLink product={props.product} path='user-guide/TBD/'>Dashboard State:</DocLink> Logical "page" or view within a dashboard. Used to build drill-down hierarchies (e.g., clicking a "Building" asset opens a state showing only its "Room" sensors).
- <DocLink product={props.product} path='user-guide/TBD/'>Widget:</DocLink> Modular UI element (chart, gauge, map, or table) used to display telemetry or execute commands.
- <DocLink product={props.product} path='user-guide/TBD/'>Widget bundle:</DocLink> Curated collection of widgets grouped by function, such as *Digital Gauges*, *Charts*, or *Control Widgets*.
