<Aside type="caution" title="Required only for legacy versions">
This page applies **only** if you are running **Trendz 1.14.0 or older** or **ThingsBoard 4.2.1 or older**. On newer versions the connection is configured automatically — no manual steps are needed.
</Aside>

## Install Trendz for ThingsBoard 4.2.1 or older

1. Follow **Steps 1–5** of the installation guide for your platform.
2. Before the **Start Trendz service** step, configure `TRENDZ_LICENSE_SECRET` and `TB_API_URL` as described below.
3. Complete all remaining steps, **except** the Sync ThingsBoard With Trendz step.
4. After the service is running, perform the <DocLink product={Products.TRENDZ} path="post-installation-steps">post-installation steps</DocLink> manually.

## Configure for your platform

Trendz requires a license secret key and the ThingsBoard API URL to connect to ThingsBoard 4.2.1 or older.
If you don't have a license key, <a href="/contact-us/" target="_blank">contact ThingsBoard support</a>.

<Tabs>
<TabItem label="Docker">

Open your `docker-compose.yml` and add the following environment variables under the `trendz` service:

```yaml
environment:
  TRENDZ_LICENSE_SECRET: YOUR_LICENSE_SECRET_HERE
  TB_API_URL: http://<thingsboard-host>:9090
```

Replace `YOUR_LICENSE_SECRET_HERE` with your license key and `<thingsboard-host>` with the address of your ThingsBoard server.

Save the file and restart the Trendz container.

</TabItem>
<TabItem label="Ubuntu">

Open the Trendz configuration file:

```bash
sudo nano /etc/trendz/conf/trendz.conf
```

Add the following lines at the end of the file:

```bash
export TRENDZ_LICENSE_SECRET=YOUR_LICENSE_SECRET_HERE
export TB_API_URL=http://<thingsboard-host>:9090
```

Replace `YOUR_LICENSE_SECRET_HERE` with your license key and `<thingsboard-host>` with the address of your ThingsBoard server (use `localhost` if ThingsBoard runs on the same machine).

Save the file and restart the Trendz service:

```bash
sudo service trendz restart
```

</TabItem>
<TabItem label="Windows">

Open the Trendz configuration file in a text editor:

```text
notepad "C:\Program Files (x86)\trendz\conf\trendz.conf"
```

Add the following lines at the end of the file:

```text
export TRENDZ_LICENSE_SECRET=YOUR_LICENSE_SECRET_HERE
export TB_API_URL=http://<thingsboard-host>:9090
```

Replace `YOUR_LICENSE_SECRET_HERE` with your license key and `<thingsboard-host>` with the address of your ThingsBoard server (use `localhost` if ThingsBoard runs on the same machine).

Save the file and restart the **Trendz** service from the Windows Services panel.

</TabItem>
</Tabs>
