## Introduction

Netvox has more than 400 LoRaWAN end devices that can be integrated into systems supporting the LoRaWAN protocol and are widely used in IoT solutions. By integrating data from Netvox devices into ThingsBoard, users can easily visualize and monitor data through the ThingsBoard <a href="/docs/pe/user-guide/dashboards/" target="_blank" rel="noopener noreferrer">dashboard</a>.

## Register Netvox devices on a third-party platform

To register a Netvox device, we will use The Things Stack Cloud Discovery. This is a free plan of <a href="https://www.thethingsindustries.com/stack/plans/" target="_blank" rel="noopener noreferrer">The Things Stack Cloud</a> that allows you to connect up to 10 devices and 10 gateways with no time restrictions, making it ideal for testing your device.

## Connect the device to ThingsBoard

<a href="/docs/pe/user-guide/integrations/" target="_blank" rel="noopener noreferrer">Integrations</a> in ThingsBoard is a feature that enables connecting external devices, services, or platforms to ThingsBoard for processing, storing, and visualizing data.

### Create integration in ThingsBoard

You need access to ThingsBoard Professional Edition. The easiest way is to use <a href="/installations/choose-region/" target="_blank" rel="noopener noreferrer">ThingsBoard Cloud</a>. Alternatively, install ThingsBoard using the <a href="/docs/pe/installation/" target="_blank" rel="noopener noreferrer">installation guide</a>.

<Steps>
1. Sign in to ThingsBoard. Go to **Integrations center > Integrations**, click the **+** icon in the upper right corner to add a new integration, and select **The Things Stack Community** as the integration type. Click **Next**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-integration-1.png',
     alt: 'ThingsBoard Integrations page with The Things Stack Community selected',
     caption: 'Integrations: selecting The Things Stack Community type and clicking Next',
   }]} />

2. Use the default Decoder function and click **Next**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-integration-2.png',
     alt: 'Integration Decoder configuration with default function',
     caption: 'Decoder step: keeping the default function and clicking Next',
   }]} />

3. Use the default Encoder function and click **Next**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-integration-3.png',
     alt: 'Integration Encoder configuration with default function',
     caption: 'Encoder step: keeping the default function and clicking Next',
   }]} />

4. Fill in the connection fields with your parameters. Enter the third-party platform's MQTT account and password, then click **Add**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-integration-4.png',
     alt: 'Integration connection settings with MQTT credentials and Add button',
     caption: 'Connection settings: entering MQTT account, password, and clicking Add',
   }]} />
</Steps>

### Create dashboard

A <a href="/docs/pe/user-guide/dashboards/" target="_blank" rel="noopener noreferrer">Dashboard</a> in ThingsBoard allows users to visualize and monitor data collected from IoT devices.

<Steps>
1. Go to **Dashboards**, click **+**, and select **Create new dashboard** from the drop-down menu. Enter the dashboard title and click **Add**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-1.png',
     alt: 'ThingsBoard Dashboards page with Create new dashboard dialog',
     caption: 'Dashboards: clicking + > Create new dashboard, entering the title, and clicking Add',
   }]} />

2. Click **+ Add widget** at the top of the screen (or the large **Add new widget** icon if this is your first widget).

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-2.png',
     alt: 'ThingsBoard empty dashboard with Add widget button',
     caption: 'Empty dashboard: clicking + Add widget to open the widget browser',
   }]} />

3. Find the **Charts** widget bundle and click on it.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-3.png',
     alt: 'Widget bundle browser with Charts bundle highlighted',
     caption: 'Widget browser: selecting the Charts bundle',
   }]} />

4. Select the **Bars** widget.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-4.png',
     alt: 'Charts bundle with Bars widget selected',
     caption: 'Charts bundle: selecting the Bars widget',
   }]} />

5. Specify the device **r718n37** as the datasource, add **Current1**, **Current2**, and **Current3** keys, then click **Add**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-5.png',
     alt: 'Bars widget configuration with r718n37 datasource and current keys',
     caption: 'Bars widget: setting r718n37 as datasource, adding Current1/Current2/Current3 keys, and clicking Add',
   }]} />

6. Drag the bottom-right corner to resize the widget, then click **Save** to save your changes.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-6.png',
     alt: 'Dashboard with Bars widget resized and Save button',
     caption: 'Dashboard: resizing the Bars widget and saving the dashboard',
   }]} />

7. Add another widget. Select **Timeseries table** to display historical data alongside the chart. Learn more about widgets in the <a href="/docs/pe/user-guide/widgets/" target="_blank" rel="noopener noreferrer">widgets documentation</a>.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/netvox/netvox-add-dashboard-7.png',
     alt: 'Dashboard with both Bars chart and Timeseries table widgets',
     caption: 'Dashboard: Bars chart and Timeseries table widgets displaying Netvox device data',
   }]} />
</Steps>

## Netvox devices

| **Model** | **Power supply** | **Turn on** |
|-----------|------------------|-------------|
| R311 Series, R312 Series, R313 Series | Battery | Short press any function key until the green and red indicator flashes once. Like R311A, R311B, R313CB, R313DA, R312A |
| R711, R315 Series | Battery | Short press the function key until the green indicator flashes once. Like R711, R31501, R315LA |
| R718 Series, R720 Series, RA08B Series | Battery | Press and hold the function key for 3 seconds until the green indicator flashes once. Like R718N37, R718H, R720A, RA08B01S |
| R718 Series, R831 Series, RA08D Series, RA07 Series, RA07xxY Series, R211 | DC in | Plug in the power adapter. Like R718N37D, R718PC, RA08D09S, RA0715, RA0716Y, R831C |
| R726 Series | Solar and rechargeable lithium batteries | Connect a battery pack. Like R72601, R72610, R72630 |
| R60 Series | DC in | Plug in the power adapter. Like R602A, R602B, R603 |
| RA02 Series | Battery | Press any function key until the green indicator flashes once. Like RA02A, RA02C |
| RA02 Series | DC in | Plug in the power adapter. Like RA02G, RA02D1 |
| RB02 Series | Battery | Press the function key until one green indicator flash and one red indicator flash. Like RB02B, RB02C, RB02I |
| R309 | Battery | Press and hold the emergency button for 3 seconds until the green/red indicator flashes once. Like R30901, R30902 |
| R719A | Battery | Hold a magnet to the top cover for 3 seconds. |
| RP02 Series | DC in | Connect a power module. Like RP02RH3PN063 |
| R809 Series | AC 100–240 V | Connect to an AC 100–240 V power source. Like R809A, R809A01 |
| R816 Series | AC 100–240 V | Connect to an AC 100–240 V power source. Like R816A, R816A01 |

More detailed information on the <a href="http://www.netvox.com.tw/index.html" target="_blank" rel="noopener noreferrer">Netvox website</a>.
