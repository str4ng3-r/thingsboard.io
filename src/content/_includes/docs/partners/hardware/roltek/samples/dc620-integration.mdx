## Introduction

This guide describes how to connect a Roltek DC620 device to ThingsBoard, configure MQTT telemetry publishing, and visualize the collected data on a dashboard.

## Create a device in ThingsBoard

<Steps>
1. Go to **Entities > Devices** and click **+** to open the add device menu.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-devices-page.png',
     alt: 'ThingsBoard Devices page',
     caption: 'ThingsBoard Devices page: opening the menu to add a new device',
   }]} />

2. Click **Add new device**, enter a name for your device, and click **Next: Credentials**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-add-device.png',
     alt: 'Add new device dialog with name field',
     caption: 'Add new device: entering the device name and proceeding to credentials',
   }]} />

3. Enable **Add credentials**, enter the desired **Access Token**, and click **Add** to save.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-device-credentials.png',
     alt: 'Device credentials dialog with Access Token field',
     caption: 'Device credentials: setting a custom Access Token and clicking Add',
   }]} />
</Steps>

## Configure the DC620

Connect the DC620 to the internet via Wi-Fi or Ethernet as described in the user manual.

### MQTT settings

<Steps>
1. Open the **MQTT Settings** page on the DC620 and configure the following:

   | Field | Value |
   |-------|-------|
   | MQTT Mode | Enable |
   | MQTT Broker URI | Your ThingsBoard server hostname or IP address |
   | MQTT Broker Username | The Access Token set in the previous step |

   Click **Save**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-mqtt-settings.png',
     alt: 'DC620 MQTT Settings page with broker URI and username fields',
     caption: 'MQTT Settings: enabling MQTT, entering the ThingsBoard broker URI and Access Token as username',
   }]} />
</Steps>

### Configure MQTT payload

<Steps>
1. On the **Rules** page, navigate to the **MQTT Payloads** tab and click **Add MQTT Payload**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-mqtt-payloads-tab.png',
     alt: 'DC620 Rules page MQTT Payloads tab with Add MQTT Payload button',
     caption: 'MQTT Payloads tab: clicking Add MQTT Payload to create a new payload template',
   }]} />

2. Configure the payload to produce JSON output compatible with ThingsBoard telemetry:

   | Field | Value |
   |-------|-------|
   | Payload Type | JSON |
   | Header | `{` |
   | Pattern | `"$V":#V` |
   | Separator | `,` |
   | Ending | `}` |

   Click **Save**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-mqtt-payload-config.png',
     alt: 'Add MQTT Payload window with JSON payload configuration fields',
     caption: 'Add MQTT Payload: configuring JSON format with header, pattern, separator, and ending fields',
   }]} />
</Steps>

### Add MQTT publisher

<Steps>
1. On the **Rules** page, navigate to the **MQTT Publishers** tab and click **Add MQTT Publisher**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-mqtt-publishers-tab.png',
     alt: 'DC620 Rules page MQTT Publishers tab with Add MQTT Publisher button',
     caption: 'MQTT Publishers tab: clicking Add MQTT Publisher to configure a new publisher',
   }]} />

2. Configure the publisher:

   | Field | Value |
   |-------|-------|
   | Period | Your desired publish interval |
   | QoS | Your preferred QoS level |
   | Retain | Your preferred retain value |
   | Topic | `v1/devices/me/telemetry` |
   | Payload | The payload created in the previous step |
   | Tags | Select the tag or tags to publish |

   Click **Save**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-mqtt-publisher-config.png',
     alt: 'Add MQTT Publisher window with topic, payload, and tag selection',
     caption: 'Add MQTT Publisher: setting the ThingsBoard telemetry topic, selecting the payload, and choosing tags to publish',
   }]} />
</Steps>

## Add data to a dashboard

<Steps>
1. Go to **Entities > Devices**, open your device, and navigate to the **Latest telemetry** tab. Click on a data row and press **Show on widget**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-latest-telemetry.png',
     alt: 'ThingsBoard device Latest telemetry tab with data row selected and Show on widget button',
     caption: 'Latest telemetry: selecting a data key and clicking Show on widget to start building a widget',
   }]} />

2. Choose a widget bundle matching your data type, select a chart, and click **Add to dashboard**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-add-widget.png',
     alt: 'Widget bundle and chart selection with Add to dashboard button',
     caption: 'Widget selection: choosing a chart type and clicking Add to dashboard',
   }]} />

3. Select **Create new dashboard**, check **Open dashboard**, and click **Add**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-create-dashboard.png',
     alt: 'Create new dashboard dialog with Open dashboard checkbox',
     caption: 'Create new dashboard: naming the dashboard and enabling Open dashboard to navigate to it immediately',
   }]} />

4. The dashboard opens with the widget displaying live telemetry from the DC620.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/roltek/roltek-dashboard-result.png',
     alt: 'ThingsBoard dashboard with widget showing DC620 telemetry data',
     caption: 'Completed dashboard: widget visualizing live telemetry data from the Roltek DC620',
   }]} />
</Steps>
