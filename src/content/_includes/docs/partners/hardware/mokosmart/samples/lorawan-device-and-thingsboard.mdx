## Introduction

This guide describes how to register a MOKO SMART LoRaWAN gateway and end devices in ThingsBoard using The Things Stack Community Edition (TTN) as the network server and a ThingsBoard MQTT integration to forward device data.

The guide applies to all MOKO SMART LoRaWAN end devices.

## Add gateway

### Add gateway on TTN

<Steps>
1. Power on the gateway and connect to its Wi-Fi access point. Open the Web GUI and note the gateway ID on the **FUNCTION - Server Access** page.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-1.png',
     alt: 'Gateway Web GUI Server Access page',
     caption: 'Gateway Web GUI: FUNCTION – Server Access page showing the gateway EUI to copy',
   }]} />

2. Log in to your TTN account and select the cluster for your frequency plan. For US915, select **North America 1**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-2.png',
     alt: 'TTN cluster selection',
     caption: 'TTN: selecting the North America 1 cluster for US915 frequency plan',
   }]} />

3. Go to the gateway console on the home page.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-3.png',
     alt: 'TTN home page gateway console link',
     caption: 'TTN home page: navigating to the gateway console',
   }]} />

4. Click **Register new gateway**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-4.png',
     alt: 'Register new gateway button',
     caption: 'TTN gateway console: clicking Register new gateway',
   }]} />

   Enter the gateway ID from step 1.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-5.png',
     alt: 'Gateway EUI entry field',
     caption: 'Pasting the gateway EUI copied from the Web GUI',
   }]} />

   Enter a TTN gateway ID, select **United States 902-928 MHz** as the frequency plan, and click **Register gateway**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-6.png',
     alt: 'Gateway registration form with ID, frequency plan, and Register button',
     caption: 'Setting the TTN gateway ID, frequency plan US 902–928 MHz, and clicking Register gateway',
   }]} />

5. Get the gateway server address from the TTN **General settings** page.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-7.png',
     alt: 'TTN gateway General settings showing server address',
     caption: 'TTN General settings: copying the gateway server address',
   }]} />

   Enter the server address in the MKGW2 Web GUI with port `1700`. Set the frequency to 915 and the channel to **US915_CH08-15** to match the TTN configuration.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-8.png',
     alt: 'MKGW2 Web GUI with server address and channel configured',
     caption: 'MKGW2 Web GUI: server address, port 1700, and channel US915_CH08-15 configured',
   }]} />

6. Verify the gateway connection. A green indicator in the Web GUI means the gateway connected successfully.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-9.png',
     alt: 'Gateway Web GUI green connected status',
     caption: 'Gateway Web GUI: green status indicator confirms successful server connection',
   }]} />

   Confirm the status on TTN as well.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-gateway-on-ttn-10.png',
     alt: 'TTN gateway overview showing Connected status',
     caption: 'TTN gateway overview: gateway status shown as Connected',
   }]} />
</Steps>

<Aside type="note">
When registering the gateway, make sure the gateway and device frequency bands match.
</Aside>

### Create integration in ThingsBoard

<Steps>
1. Go to **Integrations center** > **Integrations** and click **+** to add a new integration. Select **The Things Stack Community** as the type and click **Next**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-create-integration-in-thingsboard-1.png',
     alt: 'Add integration dialog with The Things Stack Community selected',
     caption: 'ThingsBoard Integrations: selecting The Things Stack Community type',
   }]} />

2. Paste the following decoder into the **Uplink data converter** field and click **Next**.

   ```js collapsible
   var data = decodeToJson(payload);

   var deviceName = data.end_device_ids.device_id;
   var deviceType = data.end_device_ids.application_ids.application_id;

   // If you want to parse incoming data somehow, you can add your code to this function.
   // input: bytes
   // expected output:
   //  {
   //    "attributes": {"attributeKey": "attributeValue"},
   //    "telemetry": {"telemetryKey": "telemetryValue"}
   //  }
   // default functionality - convert bytes to HEX string with telemetry key "HEX_bytes"

   function decodeFrmPayload(input) {
       var output = { attributes:{}, telemetry: {} };
       // --- Decoding code --- //

       output.telemetry.HEX_bytes = bytesToHex(input);

       // --- Decoding code --- //
       return output;
   }

   // --- attributes and telemetry objects ---
   var telemetry = {};
   var attributes = {};
   // --- attributes and telemetry objects ---

   // --- Timestamp parsing
   var incomingDateString = data.uplink_message.received_at;
   var dateString = incomingDateString.substring(0, incomingDateString.lastIndexOf(".")+3) + "Z";
   var timestamp = new Date(dateString).getTime();
   // --- Timestamp parsing

   // You can add some keys manually to attributes or telemetry
   attributes.f_port = data.uplink_message.f_port;
   attributes.settings = data.uplink_message.settings;
   // We want to save correlation ids as single object, so we are excluding them from attributes parse and add manually
   attributes.correlation_ids = data.correlation_ids;

   // You can exclude some keys from the result
   var excludeFromAttributesList = ["device_id", "application_id", "uplink_message", "correlation_ids"];
   var excludeFromTelemetryList = ["uplink_token", "gateway_id", "settings"];

   // Message parsing
   // To avoid paths in the decoded objects we passing false value to function as "pathInKey" argument.
   // Warning: pathInKey can cause already found fields to be overwritten with the last value found, e.g. receive_at from uplink_message will be written receive_at in the root.
   var telemetryData = toFlatMap(data.uplink_message, excludeFromTelemetryList, false);
   var attributesData = toFlatMap(data, excludeFromAttributesList, false);

   // Passing incoming bytes to decodeFrmPayload function, to get custom decoding
   var customDecoding = {};
   if (data.uplink_message.get("frm_payload") != null) {
       customDecoding = decodeFrmPayload(base64ToBytes(data.uplink_message.frm_payload));
   }

   // Collecting data to result
   if (customDecoding.?telemetry.size() > 0) {
       telemetry.putAll(customDecoding.telemetry);
   }

   if (customDecoding.?attributes.size() > 0) {
       attributes.putAll(customDecoding.attributes);
   }

   telemetry.putAll(telemetryData);
   attributes.putAll(attributesData);

   var result = {
       deviceName: deviceName,
       deviceType: deviceType,
       telemetry: {
           ts: timestamp,
           values: telemetry
       },
       attributes: attributes
   };

   return result;
   ```

   }]} />

3. Leave the **Downlink data converter** field empty and click **Skip**.

## Add device

<Steps>
1. In the TTN application console, go to **End devices** and click **Register end device**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-1.png',
     alt: 'TTN End devices page',
     caption: 'TTN application: End devices page with Register end device button',
   }]} />

2. Select **Enter end device specifics manually**, choose the frequency plan matching your gateway, and select **LoRaWAN version v1.0.3**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-2.png',
     alt: 'Manual device registration form',
     caption: 'Manual registration: frequency plan and LoRaWAN v1.0.3 selected',
   }]} />

3. Enter the JoinEUI. The default MOKO SMART JoinEUI is `70 B3 D5 7E D0 02 6B 87`.

4. Enter the DevEUI and AppKey. The DevEUI is printed on the package label. The default AppKey is `2B 7E 15 16 28 AE D2 A6 AB F7 15 88 09 CF 4F 3C`. You can also read both values from the device using the **MKLora** app.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-3.png',
     alt: 'JoinEUI, DevEUI, and AppKey fields',
     caption: 'Entering JoinEUI, DevEUI, and AppKey for the MOKO SMART device',
   }]} />

5. Click **Register end device**.

6. Verify the device status on TTN.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-4.png',
     alt: 'TTN device list showing registered device',
     caption: 'TTN End devices: device registered and showing online status',
   }]} />

   Check **Live data** to confirm the device is uploading payloads.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-5.png',
     alt: 'TTN Live data tab with payloads',
     caption: 'TTN Live data: device uploading payloads to the server',
   }]} />

7. Go to **Payload formatters**, paste the decoder from the <a href="https://github.com/LoRaWAN-Product-Decoder/MOKOSMART-LoRaWAN-Product-Decoder/tree/main/LW006" target="_blank" rel="noopener noreferrer">MOKO SMART LoRaWAN decoder repository</a>, and click **Save changes**.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-6.png',
     alt: 'TTN Payload formatters with MOKO SMART decoder',
     caption: 'TTN Payload formatters: MOKO SMART LW006 decoder pasted and saved',
   }]} />

8. In ThingsBoard, go to **Entities** > **Devices**. Devices registered in TTN appear automatically via the integration. Open a device and go to **Latest telemetry** to confirm data is arriving.

   <ImageGallery images={[{
     src: '/src/assets/devices-library/moko-smart/moko-smart-add-device-7.png',
     alt: 'ThingsBoard Devices page with latest telemetry',
     caption: 'ThingsBoard Devices: device created by the integration with live telemetry visible',
   }]} />
</Steps>
