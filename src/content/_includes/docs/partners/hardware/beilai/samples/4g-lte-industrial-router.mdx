## Introduction

This guide describes how to connect a KingPigeon 4G LTE Industrial Router to ThingsBoard PE using the MQTT Basic credential type.

## Configure ThingsBoard

<Steps>
1. Log in to your ThingsBoard instance and go to **Groups** > **Device Groups**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-home.png" alt="Device Groups page" />

2. Open the **All** device group and click **Add** (+) to add a new device.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-device.png" alt="Device group All" />

3. Enter the device name, for example `R40`, and click **Next: Credentials**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-device-new.png" alt="Add device" />

4. Set **Credentials type** to **MQTT Basic**. Enter a **Client ID**, **username**, and **password**, then click **Add**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-mqtt-basic.png" alt="MQTT Basic credentials" />
</Steps>

## Configure the router

<Steps>
1. Log in to the router. The default username is `admin` with no password.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-r4000.png" alt="Router login" />

2. Go to **Cloud platform**, select **Custom cloud**, and click **Add** to create a cloud configuration.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-r4001.png" alt="Cloud platform menu" />

3. Click **Edit** to edit the configuration.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-r4002.png" alt="Edit configuration" />

4. Configure the ThingsBoard connection:
   - Select **ThingsBoard** as the platform.
   - Set **MQTT ID** to the Client ID from step 4.
   - Set **Username** and **Password** from step 4.
   - Set the publishing period.
   - Click **Save**, then **Back to Overview**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-r4003.png" alt="ThingsBoard connection settings" />

5. Check **Enable setting** and click **Save & Apply**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-r4004.png" alt="Apply configuration" />
</Steps>

## Verify telemetry

After applying the router configuration, confirm that data is arriving in ThingsBoard. Click the device name to open its details, then select **Latest telemetry**.

<DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-telemetry.png" alt="Latest telemetry" />

## Create a dashboard

<Steps>
1. In **Latest telemetry**, click a data row and select **Show on widget**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-telemetry02.png" alt="Show on widget" />

2. Choose a widget bundle and a chart type, then click **Add to dashboard**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-chart.png" alt="Select widget" />

3. Create a new dashboard or select an existing one, then click **Add**.

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-chart02.png" alt="Add to new dashboard" />

   <DocImage src="/src/assets/devices-library/kingpigeon/kingpigeon-chart03.png" alt="Dashboard with widget" />
</Steps>
