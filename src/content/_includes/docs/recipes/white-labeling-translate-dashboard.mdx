This recipe shows how to apply custom translations to a dashboard title, widget titles, telemetry key labels, and axis labels using i18n keys. As an example, we translate a "Weather" dashboard from English to German.

## Prerequisites

- A ThingsBoard PE instance with <DocLink product={props.product} path="user-guide/white-labeling-translation">custom translation</DocLink> configured
- German (or another target language) added to the custom translation list

## Prepare the dashboard

Download the [Weather dashboard JSON](/docs/user-guide/resources/weather_dashboard.json) and <DocLink product={props.product} path="user-guide/dashboards#import-dashboard">import</DocLink> it into your instance.

For this example, telemetry is emulated using a <DocLink product={props.product} path="user-guide/rule-engine">Generator rule node</DocLink> with the following function:

```js
var temperature = toFixed(Math.random()*10 + 18, 2);
var humidity = toFixed(Math.random()*15 + 40, 2);
var msg = { temperature: temperature, humidity: humidity };
var metadata = { data: 40 };
var msgType = "POST_TELEMETRY_REQUEST";
return { msg: msg, metadata: metadata, msgType: msgType };
```

## Translate the dashboard title

<Steps>

1. Navigate to **White labeling** > **Custom translation** tab. Open the translation map for the target language (e.g., German).

2. Click the **plus** icon to add a new key. Enter:

   | Term key | English | German |
   |----------|---------|--------|
   | `custom.weather-dashboard.title` | Weather | Wetter |

3. Click outside the input field to save.

</Steps>

<ImageGallery images={[
  { src: '/src/assets/images/user-guide/custom-translation/internationalization-dashboard-title-1-pe.png', alt: 'Add dashboard title translation key' },
]} />

Now apply the key to the dashboard:

<Steps>

1. Navigate to **Dashboards** and open the "Weather" dashboard details.

2. Enter edit mode and replace the title with:

   ```text
   {i18n:custom.weather-dashboard.title}
   ```

3. Save the changes.

</Steps>

## Translate widget elements

Apply the same approach to widget titles, telemetry key labels, and axis labels. First, add the translation keys:

| Term key | English | German | Description |
|----------|---------|--------|-------------|
| `custom.weather-dashboard.temperature-and-humidity-widget.title` | Temperature and Humidity history | Temperatur- und Luftfeuchtigkeitsverlauf | Widget title |
| `custom.weather-dashboard.temperature.telemetry-key` | Temperature | Temperatur | Temperature key label |
| `custom.weather-dashboard.humidity.telemetry-key` | Humidity | Feuchtigkeit | Humidity key label |

<ImageGallery images={[
  { src: '/src/assets/images/user-guide/custom-translation/internationalization-widget-elements-1-pe.png', alt: 'Add widget element translation keys' },
]} />

Then assign the i18n keys in the widget configuration:

<Steps>

1. Open the "Temperature and Humidity history" widget in edit mode.

2. Set the telemetry key labels:
   - **temperature**: `{i18n:custom.weather-dashboard.temperature.telemetry-key}`
   - **humidity**: `{i18n:custom.weather-dashboard.humidity.telemetry-key}`

3. Set the Y-axis labels to the same keys.

4. Set the widget title to: `{i18n:custom.weather-dashboard.temperature-and-humidity-widget.title}`

5. Save.

</Steps>
