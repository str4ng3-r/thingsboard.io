## Introduction

<a href="https://www.ezurio.com/iot-devices/lorawan-iot-devices/rs26x-sensor" target="_blank" rel="noopener noreferrer">The Sentrius™ RS26x</a> s a battery powered IP67 temperature sensor platform that delivers HACCP level temperature monitoring with real time alerting for your food safety compliance. In addition to ensuring product safety, these intelligent devices avoid inventory loss, improve facility efficiency and save you time and money in automated day to day operations.

They are available in two variants: One with an internal temperature sensor, and a second that supports an external sensor probe.

<Banner variant="peFeature">This feature is available only for **Platform Integrations**.</Banner>

## Prerequisites

To continue with this guide we will need the following:

- <a href="https://www.ezurio.com/iot-devices/lorawan-iot-devices/rs26x-sensor" target="_blank">RS26x LoRaWAN Temperature Sensor </a>
- A smartphone with the Ezurio Xbit app for configuration of the RS26x (<a href="https://play.google.com/store/apps/details?id=com.rfpros.xbitmobile&hl=en_US" target="_blank" rel="noopener noreferrer">Android</a>/<a href="https://apps.apple.com/us/app/canvas-xbit-mobile/id6478117073" target="_blank" rel="noopener noreferrer">iOS</a>)
- LoRaWAN gateway (e.g. <a href="https://www.ezurio.com/iot-devices/lorawan-iot-devices/sentrius-rg1xx-lorawan-gateway-wi-fi-ethernet-optional-lte-us-only" target="_blank" rel="noopener noreferrer">Ezurio RG1xx</a>)
- <a href="https://www.thethingsnetwork.org/" target="_blank" rel="noopener noreferrer">The Things Stack Community Edition account</a>

- ThingsBoard account at `YOUR_TB_HOST`

## Device connection

Please follow the official RS26x Getting Started Guide: <a href="https://www.ezurio.com/documentation/quick-start-guide-connecting-rs26x-to-the-things-stack-v3" target="_blank" rel="noopener noreferrer">Connecting RS26x to The Things Stack v3</a> and the official <a href="https://www.ezurio.com/documentation/user-guide-rs26x-series" target="_blank" rel="noopener noreferrer">RS26x User Guide</a> to configure the sensor properly and connect it to The Things Stack.

## Adding the Data Converter

The first thing to add would be the uplink data converter to convert the incoming data from TTN into a usable format for Thingsboard dashboard widgets.

<Steps>

1. Navigate to **Data converter** in the **Integrations center**.

   <DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-124559.png" alt="image" />

2. Click on "**+**" and select **import converter**.

   <DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-124649.png" alt="image" />

3. Download the <a href="/docs/user-guide/resources/things_stack_community_uplink_converter_for_ezurio_rs26x.json" target="_blank" rel="noopener noreferrer" download="things_stack_community_uplink_converter_for_ezurio_rs26x.json">things_stack_community_uplink_converter_for_ezurio_rs26x.json</a> file.
4. Drag and drop it into the import window and click **Import**.
5. After the upload is complete, you should see the newly added data converter.

</Steps>

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-125249.png" alt="image" />

## Adding an Integration

Navigate to **Integration center** → **Integrations** and click "**+**" to add a new integraton.

Basic settings.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-125602.png" alt="image" />

Select Uplink Data Converter.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-125647.png" alt="image" />

Skip Downlink Data Converter.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-125708.png" alt="image" />

Enter connection details.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-130042.png" alt="image" />

Make sure to use the proper regional settings from your specific TTN settings. The credentials need to be fetched from the TTN console, see below.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-125911.png" alt="image" />

In Thingsboard click **Check connection** and you should see a **Connected** message.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-130107.png" alt="image" />

Click add to finish creating the integration.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-130156.png" alt="image" />

If you now click on the integration, you should see uplink messages coming in if the RS26x is active. Depending on the set uplink interval in your device, it might take a moment until first data can be seen.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-130334.png" alt="image" />

<Aside type="note">

The display of uplink messages is set to Failure messages only. To see all messages you need to enable this in the Debug Configuration as shown below.

</Aside>

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20251120-125900.png" alt="image" />

You can click on the three dots in the Message column and view the uplink data sent.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-130515.png" alt="image" />

The data is JSON format and looks like below. Underneath “decoded_payload” you see the RS26x battery status as well as the Temperature.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-130542.png" alt="image" />

## Create a Dashboard

Navigate to Dashboard and click "**+**".

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-133720.png" alt="image" />

This opens a new, empty dashboard where you can start adding widgets.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-133818.png" alt="image" />

Go to **Cards** widgets bundle and select a **Simple Value and chart card** widget.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-133852.png" alt="image" />

Select our device as data source and the data key should be temperature. Feel free to enter a new title as well.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20250917-152017.png" alt="image" />

You can now adjust the widget size or explore the advanced settings, for example, to add an extra decimal place.

Try adding other widgets showing other telemetry data like RSSI, SNR or the used spreading factor as indication of the link quality.

<DocImage src="/src/assets/devices-library/ready-to-go-devices/s26x-lorawan/image-20251110-103326.png" alt="image" />
